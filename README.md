# DBMS Laboratory – MySQL

This repository contains structured SQL scripts and documentation for the
**Database Management Systems Laboratory (21CSC205P)**.

## Tech Stack
- Database: MySQL 8.0+
- Language: SQL, PL/SQL
- Tools: MySQL Workbench / VS Code

## Experiments Covered
1. DDL and DML Commands
2. DQL Commands
3. TCL and DCL Commands
4. Relational Algebra & Tuple Relational Calculus
5. Subqueries, Joins, and Views
6. PL/SQL – Procedures, Functions, Cursors, Triggers
7. Normalization (1NF to 5NF)
8. Transaction & Concurrency Control

## How to Run
```sql
CREATE DATABASE dbms_lab;
USE dbms_lab;
```
1. Execute schema files from /schemas

2. Insert data from /sample-data

3. Run experiment scripts from /experiments
